# 商店功能演示

## 功能概述

我已经为"烽尘绘谱"游戏实现了完整的商店功能，包括以下特性：

### 1. 商店主界面 (`lib/screens/shop_screen.dart`)
- **分类浏览**: 推荐、武器、防具、饰品、消耗品五个分类
- **商品展示**: 显示物品图标、名称、描述、属性加成、稀有度
- **价格显示**: 金币价格和库存信息
- **购买功能**: 支持单个或批量购买（消耗品）
- **推荐系统**: 根据玩家等级智能推荐合适的装备

### 2. 购买历史 (`lib/screens/purchase_history_screen.dart`)
- **历史记录**: 显示所有购买记录，按时间倒序排列
- **详细信息**: 物品名称、数量、总价、购买时间
- **统计入口**: 可以跳转到购买统计页面

### 3. 购买统计 (`lib/screens/shop_statistics_screen.dart`)
- **总体统计**: 总消费、购买物品数、交易次数、平均单价
- **分类统计**: 各类物品的购买比例
- **热门商品**: 最受欢迎的商品排行榜

### 4. 商店服务 (`lib/services/shop_service.dart`)
- **推荐算法**: 基于玩家等级的智能推荐
- **购买验证**: 金币和库存检查
- **统计分析**: 购买数据的深度分析

## 商品系统

### 稀有度系统
- **普通** (灰色): 基础装备，价格便宜
- **稀有** (蓝色): 中等装备，属性较好
- **史诗** (紫色): 高级装备，属性优秀
- **传说** (橙色): 顶级装备，属性极佳

### 商品类型
1. **武器**: 提升攻击力和速度
2. **防具**: 提升防御力，部分影响速度
3. **饰品**: 提升各种属性的平衡装备
4. **消耗品**: 恢复生命值、提供经验等

### 库存系统
- **无限库存**: 基础商品，标记为 -1
- **限量商品**: 稀有装备，库存有限
- **库存刷新**: 支持手动刷新商店库存

## 数据持久化

使用 `SharedPreferences` 保存：
- 商店库存状态
- 购买历史记录
- 玩家金币和背包物品

## 用户体验

### 界面设计
- **一致的主题**: 使用游戏统一的金色主题
- **直观的图标**: 每个物品都有对应的表情符号图标
- **清晰的信息**: 价格、属性、库存一目了然
- **流畅的动画**: 购买确认对话框和状态更新

### 交互设计
- **购买确认**: 防止误操作的确认对话框
- **数量选择**: 消耗品支持批量购买
- **即时反馈**: 购买成功/失败的即时提示
- **状态同步**: 购买后立即更新金币和库存

## 集成说明

商店功能已完全集成到主游戏中：

1. **主屏幕导航**: 从主屏幕可以直接进入商店
2. **背包系统**: 购买的物品自动添加到背包
3. **装备系统**: 可以直接装备购买的装备
4. **金币系统**: 与游戏进度系统完全同步

## 扩展性

代码设计考虑了未来扩展：
- **折扣系统**: 预留了折扣价格计算
- **限时商品**: 支持限时特价商品
- **VIP系统**: 可以轻松添加会员折扣
- **活动商品**: 支持节日或活动特殊商品

## 使用方法

1. 启动游戏后，从主屏幕点击"商店"
2. 浏览不同分类的商品
3. 点击"购买"按钮购买心仪的装备
4. 查看购买历史和统计数据
5. 在背包中装备购买的物品

商店功能现在已经完全可用，为玩家提供了丰富的装备获取途径！
