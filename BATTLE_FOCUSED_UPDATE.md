# 烽尘绘谱 - 回合对战游戏更新

## 🎮 核心改动

根据用户需求，已将游戏从AVG（冒险游戏）模式改为以**回合对战**为核心的游戏模式。

### 主要变更

#### 1. 征程系统重构 ⭐ 核心更新
- **移除**: AVG式的剧情选择和文本冒险
- **新增**: 关卡制战斗挑战系统
- **特色**: 
  - 5个经典三国战役关卡
  - 渐进式难度设计（1-5星）
  - 关卡解锁机制
  - 丰富的战斗奖励

#### 2. 关卡设计
```
第一章 - 群雄逐鹿
├── 1-1 黄巾起义 (★☆☆☆☆) - 张角、张宝、张梁
├── 1-2 讨伐董卓 (★★☆☆☆) - 董卓、吕布、李傕  
├── 1-3 虎牢关之战 (★★★☆☆) - 吕布、张辽、高顺
├── 1-4 官渡之战 (★★★★☆) - 袁绍、颜良、文丑
└── 1-5 赤壁之战 (★★★★★) - 曹操、张辽、徐晃
```

#### 3. 战斗系统增强
- **自定义敌军**: 每个关卡都有独特的敌军配置
- **历史武将**: 13位三国名将作为BOSS
- **平衡设计**: 根据历史地位设计武将属性
- **奖励机制**: 经验、金币、装备奖励

## 🎯 游戏流程

### 新的核心循环
1. **选择关卡** → 查看敌军配置和奖励
2. **准备战斗** → 调整阵型和装备
3. **回合对战** → 策略性战斗
4. **获得奖励** → 经验、金币、装备
5. **武将成长** → 升级强化
6. **解锁新关卡** → 挑战更强敌人

### 关卡特色

#### 黄巾起义 (入门关卡)
- **敌军**: 张角(谋士) + 张宝、张梁(前锋)
- **特点**: 教学关卡，难度较低
- **奖励**: 基础装备和经验

#### 讨伐董卓 (进阶关卡)  
- **敌军**: 董卓(统帅) + 吕布(超强前锋) + 李傕
- **特点**: 吕布武力值100，需要策略应对
- **奖励**: 精铁装备

#### 虎牢关之战 (挑战关卡)
- **敌军**: 吕布 + 张辽、高顺
- **特点**: 全前锋阵容，攻击力极强
- **奖励**: 稀有装备

#### 官渡之战 (困难关卡)
- **敌军**: 袁绍(统帅) + 颜良、文丑(猛将)
- **特点**: 兵力优势，需要以智取胜
- **奖励**: 传说装备

#### 赤壁之战 (终极关卡)
- **敌军**: 曹操(智力95) + 张辽、徐晃
- **特点**: 智勇双全的最强组合
- **奖励**: 顶级装备

## 🎨 界面设计

### 征程界面重设计
- **关卡卡片**: 每个关卡独立卡片显示
- **进度指示**: 清晰的完成状态和解锁状态
- **敌军预览**: 显示敌军武将头像
- **奖励展示**: 预览战斗奖励
- **难度标识**: 星级难度系统

### 视觉元素
- **状态图标**: 
  - ✅ 已完成 (绿色)
  - ⚔️ 可挑战 (金色)  
  - 🔒 未解锁 (灰色)
- **敌军头像**: 红色圆形头像显示敌将
- **难度星级**: 1-5星难度可视化

## 🔧 技术实现

### 数据结构
```dart
class BattleStage {
  final String id;           // 关卡ID
  final String name;         // 关卡名称  
  final String description;  // 关卡描述
  final int difficulty;      // 难度等级(1-5)
  final List<General> enemies; // 敌军配置
  final BattleRewards rewards; // 奖励配置
  final bool isCompleted;    // 完成状态
  final bool isUnlocked;     // 解锁状态
}
```

### 战斗系统扩展
- **自定义敌军**: 支持传入特定敌军配置
- **胜利回调**: 战斗结果返回给关卡系统
- **进度保存**: 关卡完成状态持久化

### 平衡性设计
- **属性设计**: 基于历史地位设计武将属性
- **难度曲线**: 渐进式难度提升
- **奖励平衡**: 难度与奖励成正比
