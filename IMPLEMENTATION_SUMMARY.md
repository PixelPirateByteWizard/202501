## 核心功能流程

### 升级检查流程
1. 检查武将经验是否达到上限
2. 计算升级所需材料（基于等级和稀有度）
3. 检查玩家是否拥有足够材料和银币
4. 返回检查结果

### 升级执行流程
1. 显示升级预览对话框
2. 用户确认后执行升级
3. 消耗相应材料和银币
4. 提升武将等级和属性
5. 保存游戏状态
6. 显示升级成功提示

### 材料需求计算
- **银币**：100 + (等级 × 50) + (稀有度 × 100)
- **升级石**：根据等级选择类型，数量递增
- **经验卷轴**：根据稀有度确定类型和数量
