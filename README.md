# 西游幻途

## 应用简介

一款基于中国古典名著《西游记》改编的互动冒险游戏。玩家将跟随唐僧师徒四人，踏上充满挑战与奇遇的西天取经之路。

### 游戏特色

- **丰富的剧情体验**：重现西游记经典故事，包括三打白骨精、真假美猴王、无字经书等精彩章节
- **互动式选择**：每个剧情节点提供多个选择，玩家的决策将影响故事发展方向
- **AI驱动剧情**：采用先进AI技术生成动态剧情，确保每次游戏体验的新鲜感
- **精美的视觉设计**：中国传统风格的界面设计，配以精美的背景图和角色形象
- **多样化角色系统**：包含孙悟空、猪八戒、沙和尚等经典角色，每个角色拥有独特能力

### 游戏玩法

1. 选择游戏关卡，进入不同的西游记故事章节
2. 阅读剧情发展，在关键节点做出选择
3. 正确的选择将推动故事向原著方向发展
4. 完成章节挑战，解锁新的游戏内容

### 技术特点

- 基于Flutter框架开发，提供流畅的跨平台体验
- 集成AI服务，实现动态剧情生成
- 本地数据存储，保存游戏进度和玩家选择

### 开发团队

由热爱中国传统文化的开发者精心打造，致力于通过现代交互方式传递经典文学魅力。

## 安装与运行

flutter pub get
