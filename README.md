ChromaPour: Color Sort - App Overview

ChromaPour is a liquid sorting puzzle game developed using Flutter, offering a simple yet challenging gameplay experience where players need to sort mixed colored liquids into different bottles. The game features multiple progressively difficult levels, beautiful UI design, and smooth animations, providing an enjoyable and immersive gaming experience.

Key Features
Core Gameplay: Players must sort mixed colored liquids into bottles, with each bottle having limited capacity, testing their logical thinking.
Level System: Multiple levels with increasing difficulty, challenging the player's problem-solving and strategy skills.
Assistive Features: Includes features like undo, add empty bottles, and eliminate liquids to help players solve puzzles.
Progress Save: Automatically saves the game progress and best scores, allowing players to continue their game anytime.
Beautiful UI: A simple, elegant interface with smooth animations enhancing the overall user experience.
Responsive Design: The game interface adapts to various screen sizes, ensuring smooth performance on different devices.
