# 烽尘绘谱 - 功能特性总览

## 🎮 核心功能

### 1. 主界面 (Home Screen)
- **功能菜单导航**: 6个主要功能模块的入口
- **游戏进度显示**: 当前章节、武将数量、玩家等级
- **数据统计**: 胜利场次、失败场次、招募武将数等
- **自动数据加载**: 启动时自动加载或创建新游戏

### 2. 征程系统 (Journey Screen)
- **AI驱动剧情**: 使用DeepSeek AI实时生成剧情文本
- **动态选择系统**: 玩家选择影响后续剧情发展
- **章节进度管理**: 支持多章节、多节点的故事结构
- **战斗触发**: 特定选择可触发战斗场景

### 3. 武将系统 (Generals Screen)
- **武将管理**: 查看所有招募的武将
- **属性展示**: 武力、智力、统率、速度、兵力五维属性
- **等级系统**: 武将可通过经验升级，提升属性
- **技能系统**: 每个武将拥有独特的主动/被动技能
- **装备系统**: 武将可装备武器、防具、饰品
- **战力计算**: 综合属性、装备、等级的总战力评估
- **分类筛选**: 按职业（前锋、谋士、辅助）筛选武将

### 4. 阵型系统 (Formation Screen)
- **拖拽布阵**: 直观的拖拽操作布置武将位置
- **多种阵型**: 锋矢阵、雁行阵、鱼鳞阵等经典阵型
- **位置加成**: 不同位置提供不同的属性加成
- **阵型切换**: 可随时更换不同的战术阵型
- **实时保存**: 阵型配置自动保存到游戏数据

### 5. 背包系统 (Inventory Screen)
- **装备管理**: 查看和管理所有装备
- **分类浏览**: 装备、道具、材料分类显示
- **装备详情**: 详细的装备属性和特殊效果说明
- **一键装备**: 选择武将直接装备物品
- **品质系统**: 1-5星品质等级，不同颜色区分

### 6. 商店系统 (Shop Screen) ⭐ 新增
- **装备商店**: 购买各种品质的武器、防具、饰品
- **金币系统**: 使用金币购买物品
- **价格计算**: 基于装备品质和属性的动态定价
- **库存管理**: 购买后物品自动添加到背包

### 7. 成就系统 (Achievement Screen) ⭐ 新增
- **多类型成就**: 战斗、收集、剧情、武将、等级成就
- **进度追踪**: 实时显示成就完成进度
- **奖励系统**: 完成成就获得金币、经验等奖励
- **分类筛选**: 按成就类型筛选查看

### 8. 战斗系统 (Battle Screen)
- **回合制战斗**: 经典的回合制战斗机制
- **AI战斗报告**: 实时生成文学化的战斗描述
- **技能系统**: 武将可使用独特技能
- **阵型影响**: 阵型配置影响战斗表现
- **经验奖励**: 战斗胜利获得经验值提升武将等级
- **沙盘视觉**: 简洁的战场态势显示

### 9. 设置系统 (Settings Screen)
- **音画设置**: 背景音乐、音效音量、文本速度调节
- **AI偏好**: 叙事风格（正史/演义/白话）、生成速度设置
- **游戏选项**: 推送通知、战斗动画开关
- **数据管理**: 游戏数据重置功能

## 🔧 技术特性

### 数据持久化
- **SharedPreferences**: 本地数据存储
- **JSON序列化**: 完整的游戏状态保存/加载
- **自动保存**: 关键操作自动保存游戏进度

### AI集成
- **DeepSeek API**: 集成DeepSeek AI生成动态内容
- **智能缓存**: 减少API调用，提升响应速度
- **备用机制**: API失败时使用预设内容

### 音效系统
- **触觉反馈**: 使用系统震动模拟音效
- **音效分类**: 按钮点击、战斗、升级等不同音效
- **音量控制**: 独立的音效和音乐音量控制

### 用户体验
- **Material Design**: 遵循Material Design设计规范
- **深色主题**: 古典风格的深色主题
- **响应式布局**: 适配不同屏幕尺寸
- **流畅动画**: 页面切换和交互动画

## 📱 用户流程

### 新用户流程
1. 启动应用 → 自动创建新游戏
2. 获得初始武将（关羽、张辽、郭嘉）
3. 获得初始装备和金币
4. 进入征程开始游戏

### 核心游戏循环
1. **征程** → 推进剧情，做出选择
2. **战斗** → 参与回合制战斗
3. **奖励** → 获得经验、装备、金币
4. **成长** → 升级武将、购买装备
5. **布阵** → 调整阵型和武将配置
6. 重复循环
