# 增强战斗系统演示

## 系统概述

我已经完全重新设计了战斗系统，解决了之前的所有问题，并添加了许多新功能。新的战斗系统分为三个阶段：准备阶段、战斗阶段和结束阶段。

## 🎯 解决的问题

### 1. 武将选择问题 ✅
- **之前**: 无法选择武将，只能使用前3个武将
- **现在**: 可以从所有武将中选择最多6个武将参战
- **功能**: 点击武将头像进行选择/取消选择

### 2. 操作冲突问题 ✅
- **之前**: 攻击、技能、开始战斗按钮同时存在，逻辑混乱
- **现在**: 分阶段操作，准备阶段选择武将，战斗阶段进行操作
- **功能**: 先选择操作的武将，再选择攻击或技能

### 3. 阵型应用问题 ✅
- **之前**: 阵型系统存在但没有实际应用
- **现在**: 阵型效果在准备阶段显示，影响总战力计算
- **功能**: 显示阵型加成效果，应用到战斗力计算

### 4. 多英雄上阵问题 ✅
- **之前**: 只能上阵3个武将
- **现在**: 最多可以上阵6个武将
- **功能**: 灵活的武将选择和战力计算

## 🎮 新战斗系统流程

### 阶段1: 准备阶段
1. **阵型信息显示**
   - 显示当前使用的阵型
   - 展示阵型效果和属性加成
   - 阵型加成应用到总战力

2. **武将选择**
   - 网格显示所有可用武将
   - 点击选择/取消选择武将
   - 最多选择6个武将参战
   - 实时显示选择状态

3. **出战武将预览**
   - 显示已选择的武将列表
   - 展示每个武将的详细信息
   - 计算并显示总战力
   - 可以移除已选择的武将

4. **开始战斗**
   - 必须至少选择1个武将
   - 点击"开始战斗"进入战斗阶段

### 阶段2: 战斗阶段
1. **战场显示**
   - 我军vs敌军生命值对比
   - 当前选中武将高亮显示
   - 回合数显示

2. **武将操作选择**
   - 横向滑动选择要操作的武将
   - 选中的武将有特殊边框高亮
   - 显示当前操作的武将

3. **战斗操作**
   - **普通攻击**: 造成基础伤害
   - **释放技能**: 使用武将技能，伤害更高
   - **结束回合**: 结束我方回合，敌军反击

4. **战斗记录**
   - 实时显示战斗过程
   - 记录伤害数值和技能使用
   - 滚动显示历史记录

### 阶段3: 结束阶段
1. **胜利结算**
   - 显示战斗结果
   - 展示获得的奖励
   - 领取奖励并返回

2. **失败处理**
   - 显示失败信息
   - 提供重新挑战选项
   - 返回准备阶段重新配置

```dart
          Chip(label: Text('${bonus.key} +${bonus.value}%'))
        ),

    ],

```

### 战力计算
```dart
int _calculateTotalPower() {
  int totalPower = 0;
  for (final general in _selectedGenerals) {
    totalPower += _calculateGeneralPower(general);
  }
  
  // 应用阵型加成
  if (_currentFormation != null) {
    for (final bonus in _currentFormation!.bonuses.entries) {
      if (bonus.key == 'attack') {
        totalPower = (totalPower * (1 + bonus.value / 100)).round();
      }
    }
  }
  
  return totalPower;
}
```

## 📊 数据统计

### 战力计算公式
```
武将战力 = 攻击 + 防御 + 智力 + 速度
总战力 = Σ(武将战力) × (1 + 阵型加成%)
```

### 伤害计算
- **普通攻击**: 10-30点随机伤害
- **技能攻击**: 15-45点随机伤害
- **敌军反击**: 5-20点随机伤害
